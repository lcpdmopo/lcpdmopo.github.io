# Intelligence Association Site Clone (Clean-Room Starter)

This repository is a **clean-room, original-code recreation** of the public structure and visual patterns of a traditional membership association website similar to AFIO.

It is **not** a verbatim copy of any proprietary site assets, branding, or protected text. Replace the placeholder copy, logos, and imagery with material you have the rights to use.

## Included

- Responsive homepage
- Expandable multi-level navigation
- Hero / news carousel section
- News cards
- Membership section
- Events section
- Sponsor grid
- Footer with contact block
- Simple vanilla JS (no framework required)

## Run locally

Because this is a static site, you can open `index.html` directly, or serve it locally:

### Python

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy

This repo can be deployed easily to:

- GitHub Pages
- Netlify
- Vercel (static)
- Cloudflare Pages

## Notes

- The content in `script.js` is data-driven and easy to replace.
- Colors, typography, and layout are intentionally designed to resemble a formal nonprofit / professional association website.
- For a true production clone of a site you own, you would still want to add CMS integration, forms, auth, event feeds, and asset licensing.
